Typography

The current visual style appears to use a clean sans-serif system font.

Font Family

Recommended:

font-family: system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;

Type Scale
Element	Size	Weight	Notes
Login app title	1.8em	700	Centered
App header title	1.1em	700	Indigo
Dashboard greeting	1.6em	700	Strong visual anchor
Section heading	1.3em	600	Used for cards
Task title	1.2em	600–700	Indigo
Body text	1em	400	Gray
Metadata	0.75em	400–600	Due dates, subject tags
Settings section label	0.82em	800	Uppercase, letter-spaced
Form labels	0.78em	800	Uppercase, letter-spaced

Typography Rules

Use uppercase letter-spaced labels for form and settings labels:

text-transform: uppercase;
letter-spacing: 0.12em;
font-weight: 800;

Use blue or indigo for headings, but avoid applying the primary color to all text. Keep body text neutral for readability.

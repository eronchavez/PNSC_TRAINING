# React Offline Starter Setup

This starter project is prepared for the Web Technologies competition environment.

The project can run without internet access because all required packages have already been installed in this folder.

## 1. Available Technologies

This starter includes:

- React
- Vite
- React Router
- Lucide React icons
- Tailwind CSS
- Bootstrap

Competitors may choose either:

1. Tailwind CSS
2. Bootstrap
3. Plain custom CSS

Use only one main styling approach unless the competition task specifically requires otherwise.

## 2. Important Offline Rule

Internet access is not available during the competition.

Do not run commands that require downloading packages from the internet, such as:

```bash
npm install package-name
npm create vite@latest
npx create-react-app
npx shadcn
npm update
